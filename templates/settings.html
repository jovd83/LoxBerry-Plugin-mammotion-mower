State topics: /<device>/<datapoint>  ·  Health: /_status, _device_count, _last_poll_epoch

Writes <plugin config>/mqtt_subscriptions.cfg so the built-in MQTT Gateway relays /# to the Loxone Miniserver as Virtual Inputs.

Commands (Loxone → mower)

Loxone publishes a payload (any string — the command name is in the topic) to /<device>//<command>. Supported commands: start_task, pause, resume, cancel, return_to_dock, leave_dock, sync_maps, restart_mower, move_forward, move_back, move_left, move_right.

" placeholder="set" style="max-width: 200px;">

Diagnostics
